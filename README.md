## ✨ Overview

Visthara is a premium luxury resort website built with modern web technologies. Experience breathtaking visuals, smooth animations, and an intuitive interface that showcases the epitome of luxury travel and accommodation.

## 🚀 Features

- **🏨 Luxury Accommodations**: Explore exquisite stays with detailed descriptions and stunning imagery
- **🎯 Unique Experiences**: Discover curated experiences that create unforgettable memories
- **📸 Interactive Gallery**: Immerse yourself in our visual journey through the resort
- **📝 Easy Booking**: Seamless booking process with modern UI components
- **🎨 Cinematic Animations**: Smooth, professional animations powered by Framer Motion and GSAP
- **📱 Responsive Design**: Perfect experience across all devices
- **⚡ Fast Performance**: Built with Vite for lightning-fast development and builds

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite
- **Styling**: Tailwind CSS, PostCSS
- **Animations**: Framer Motion, GSAP
- **Icons**: Lucide React
- **Carousel**: Swiper
- **Linting**: ESLint
- **Build Tool**: Vite

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/madansp7676/Vistara.git
   cd Vistara
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` to view the application.

## 🏗️ Build for Production

```bash
npm run build
```

## 🧪 Linting

```bash
npm run lint
```
